# Webマーケのデモリポジトリへようこそ
このリポジトリは、GitHubの便利な機能をシンプルにご紹介するためのデモ用プロジェクトです。

---

## 制作の流れ

### 1. ブランチを作成する
`main` ブランチから作業用ブランチを切ります。ブランチ名は担当者名やページ名を含めると管理しやすくなります。

```bash
git checkout -b 担当者名_作業内容
```

### 2. ファイルを編集する
HTMLやCSS、画像など必要なファイルを追加・編集します。

### 3. 変更をコミットする
編集内容をこまめにコミットして、変更の履歴を残します。

```bash
git add .
git commit -m "変更内容のメモ"
```

### 4. プルリクエストを作成する
作業が完了したら GitHub 上でプルリクエスト（PR）を作成します。  
変更内容や確認してほしい点をコメントに記載します。

### 5. レビュー・フィードバック
担当者がコードや内容を確認し、必要に応じて修正依頼を出します。

### 6. `main` にマージする
レビューが通ったら `main` ブランチにマージして、本番反映の準備完了です。
