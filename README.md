# Webマーケのデモリポジトリへようこそ
このリポジトリは、GitHubの便利な機能をシンプルにご紹介するためのデモ用プロジェクトです。

---

## サイト制作の流れ

### 1. ブランチを作成する
`main` ブランチから、担当者名や案件名をつけた作業ブランチを作成します。

```bash
git checkout -b your_branch_name
```

### 2. ファイルを編集する
HTMLやCSS、画像などのファイルを追加・編集します。

### 3. 変更をコミットする
編集内容をコミットして、変更履歴を記録します。

```bash
git add .
git commit -m "変更内容の説明"
```

### 4. プッシュする
作業ブランチをリモートリポジトリ（GitHub）に送信します。

```bash
git push origin your_branch_name
```

### 5. プルリクエストを作成する
GitHub上でプルリクエスト（PR）を作成し、担当者にレビューを依頼します。

### 6. レビュー・修正
レビュアーがコードを確認し、必要であれば修正を行います。

### 7. `main` にマージする
レビューが承認されたら、`main` ブランチにマージして制作完了です。

以下のプロンプトをまとめて投げ、ページの骨格を一気に生成しました。

> あなたはプロフェッショナルなWebデザイナー兼フロントエンドエンジニアです。
> 以下の要件と【指定テキスト】を必ず使用して、若者向けのカジュアルかつ少し贅沢な「POPなお鮨屋さん」のトップページ（ランディングページ）を作成してください。
>
> HTML、CSS、JavaScriptを使用して、1つのファイルで動作するように出力してください。（CSSやJSはファイル内に埋め込んでください）
> AI自身で長い文章を生成する必要はありません。提供されたテキストを適切にマークアップ・デザインすることに集中してください。
>
> **ターゲットとデザインの方向性**
> - ターゲット：20代〜30代。給料日のご褒美やデートなど、吉祥寺でのプチ贅沢。

> - レイアウト：写真をアシンメトリーに配置し、余白を大きくとって洗練された抜け感を出す。
> - テキスト：見出しやキャッチコピーには「縦書き」を取り入れ、和の雰囲気を出す。
> - 遊び心（重要）：全体のベースは上品にしつつ、英語の見出し（Concept, Menuなど）や一部のキャッチコピーに「ポップな丸ゴシックや手書き風フォント（Google Fontsの 'Kiwi Maru' や 'Zen Maru Gothic' など）」を使用し、若者らしいPOPさを演出する。

### Step 2｜画像の差し替え

初回生成ではUnsplashやダミー画像URLを使用していましたが、その後 **実際の料理写真を `image/` フォルダに格納** し、HTML内の `src` を以下のように差し替えました。

| 変数名（仮） | ファイル名 | 使用箇所 |
|---|---|---|
| ヒーロー背景 | `image/food_1.jpg` | Hero セクション背景 |
| コンセプト画像 | `image/food_6.jpg` | Concept セクション右カラム |
| メニュー1 | `image/food_5.jpg` | おまかせ握りカード |
| メニュー2 | `image/food_4.jpg` | こぼれウニ小鉢カード |
| メニュー3 | `image/food_3.jpg` | 日本酒ペアリングカード |
| ギャラリー | `image/food_1〜6.jpg` | Gallery グリッド |

> **ポイント：** 画像は `image/` フォルダにまとめて置く。ファイル名はわかりやすい名前にしておくと差し替えがラク。

## index.html の作り方

### ファイル構成
```
web-demo-repository/
├── index.html       ← 1ファイルにHTML・CSS・JSをまとめた構成
└── image/
    ├── food_1.jpg
    ├── food_2.jpg
    ├── food_3.jpg
    ├── food_4.jpg
    ├── food_5.jpg
    └── food_6.jpg
```

### ページ構成（セクション）

| セクション | 役割 |
|---|---|
| `<nav>` | 固定ナビゲーションバー（スクロールで背景が変わる） |
| `.hero` | ファーストビュー（背景画像＋縦書きキャッチコピー） |
| `#concept` | コンセプト紹介（テキスト＋画像の2カラム） |
| `#menu` | メニュー一覧（カード形式・3列グリッド） |
| `#gallery` | フォトギャラリー（4列の非対称グリッド） |
| `#access` | アクセス情報（住所・営業時間＋地図エリア） |
| `<footer>` | フッター |
| `.fixed-cta` | 画面下部に固定された「Web予約」ボタン |

### CSS のポイント

**カラー変数**（`:root` でまとめて管理）
```css
--navy:  #0b1929   /* 背景の濃紺 */
--gold:  #c8a456   /* アクセントのゴールド */
--white: #f5f0e8   /* 温かみのある白 */
```

**フォント**（Google Fonts から読み込み）
- `Noto Serif JP` … 本文・見出し（和文）
- `Kiwi Maru` … ラベル・英字
- `Zen Maru Gothic` … 補助テキスト

**スクロールアニメーション**
`.reveal` クラスを要素に付けておき、JavaScriptで `.visible` を追加して
`opacity` と `translateY` をトランジションさせる仕組み。

### JavaScript のポイント

1. **ヒーロー画像のズームアウト** … ページ読み込み時に `zoom-out` クラスを付与してゆっくり縮小
2. **ナビの背景切り替え** … `scroll` イベントで `solid` クラスを付与・削除
3. **スクロール登場アニメーション** … `IntersectionObserver` で `.reveal` 要素を監視し、画面に入ったら `.visible` を付与

### レスポンシブ対応

| 幅 | 変化 |
|---|---|
| `≤ 960px` | ナビのリンクを非表示、2カラム → 1カラムに変更 |
| `≤ 520px` | フォントサイズ縮小、バッジ非表示 |
