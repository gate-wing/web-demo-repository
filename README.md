# Webマーケのデモリポジトリへようこそ
このリポジトリは、GitHubの便利な機能をシンプルにご紹介するためのデモ用プロジェクトです。

## 制作の流れ

### 1. 企画・設計
- クライアントへのヒアリング（目的・ターゲット・要件の整理）
- サイトマップ・ワイヤーフレームの作成
- デザインカンプの作成・確認

### 2. 環境構築
- GitHubリポジトリの作成
- ブランチ運用ルールの決定（`main` / `feature/*` など）
- フォルダ構成の準備（`css/` `js/` `img/` など）

### 3. コーディング
- HTML：セクションごとにマークアップ（ヘッダー、ヒーロー、コンセプト、メニュー、店舗情報、フッター）
- CSS：デザインカンプに沿ったスタイリング・レスポンシブ対応
- JavaScript：ハンバーガーメニュー、スクロールアニメーションなどのインタラクション実装

### 4. レビュー・修正
- プルリクエストを作成し、チームメンバーがコードレビュー
- フィードバックをもとに修正・再レビュー
- デザインとの差異チェック、ブラウザ・デバイステスト

### 5. 公開・納品
- `main` ブランチへマージ
- 本番環境へデプロイ
- クライアントへの納品・公開確認
